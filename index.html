课前提醒

三种方式，推荐第三种（订阅 URL）。

手机上点开直接导入。⚠️ OPPO 日历会把所有事件归到"默认日程"，不能分类。

③ 订阅 URL（推荐） — OPPO 日历会显示成独立分类，课表更新时自动同步：

  1. 下载下面的 calendar.ics 文件
  2. 把它放到你 GitHub 仓库根目录，push 到 GitHub
  3. OPPO 日历 → 日程同步与导入 → URL 日历 → 粘贴：
    https://<你的GitHub用户名>.github.io/class-schedule/calendar.ics
